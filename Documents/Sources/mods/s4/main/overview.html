The mod Main may well be an important toolkit that another mod are thinking about using relies on. If another mod says it requires Main, you must download it or it may not function.

Mod creators should check out the documentation for a bit of explanation of the parts of this mod. The source code is also available for use as a reference in the 'files' tab.

Note: The features listed below are mostly directed toward creators. If you are not a mod creator, the big feature of Main is that it allows other mods to function.

Features

  • Optionally allow Main to control the loading of your mod, which will permit greater control over how it happens.
  • Easy to use system for patching functions.
  • Tools for saving simple persistent data such as settings.
  • Automatically register interactions to game objects through tuning.
  • Allows you to reference localization strings with named identifiers instead of just numbers in certain situations.
